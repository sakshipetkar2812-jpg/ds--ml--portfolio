Zomato Restaurant Analysis
Project Overview

This project analyzes Zomato restaurant data to understand customer preferences, ratings, and spending patterns. The analysis aims to provide insights for improving customer experience and guiding restaurant promotions.

Key Questions Explored

Most Popular Restaurant Type
Identify the type of restaurant from which the majority of customers place orders. This helps understand customer preferences and trends.

Customer Votes by Restaurant Type
Determine how many votes each type of restaurant has received from customers. This indicates the popularity and engagement level for each type.
