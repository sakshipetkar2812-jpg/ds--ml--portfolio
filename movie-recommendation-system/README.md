## Description
This project is a content-based movie recommendation system built using Python and Machine Learning.
It recommends similar movies based on metadata such as genres, keywords, cast, director, and overview.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Features
- Content-based filtering
- Cosine similarity
- Text vectorization using CountVectorizer
- Real-time movie recommendations

## Project Structure
movie-recommendation-system/
- movie_recommendation_system.ipynb
- data/movies.csv

## How to Run
1. Clone the repository  
2. Open Jupyter Notebook  
3. Run movie_recommendation_system.ipynb  
4. Enter a movie name to get recommendations

## Example
Input: Avatar  
Output: Similar movies recommended
